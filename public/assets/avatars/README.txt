# Avatar Images for Chatgjipito

Place your 6 avatar images in this folder with the following names:
- 1.png
- 2.png
- 3.png
- 4.png
- 5.png
- 6.png

Recommended specifications:
- Square aspect ratio (1:1)
- Size: at least 256x256 pixels
- Format: JPG or PNG (if using PNG, rename them to .jpg)
- Albanian-themed avatars would enhance the experience

These images will be randomly selected for each AI response in the chat interface.
